Комплекс VR-тренажеров "Рудник"

В рамках данного проекта перед нашей командой стояла задача разработки виртуального комплекса тренажеров, ориентированных на формирование практических навыков по выполнению опасных производственных операций в условиях рудничной шахты.

  • Released: 2023
  • Platforms: Windows (Oculus Quest | HTC VIVE)
  • Team Size: four core members
  • Duration: 12 months
  • Responsibilities: руководство проектом, геймдизайн, разработка, дизайн уровней
  • Tools Used: Unreal Engine 4, C++, Blueprints, Git

Description

Detailed Info

Одной из ключевых особенностей комплекса стало использование различных средств управления. В дополнение к классическому виртуальному решению мы интегрировали целый набор реальных физических пультов, созданных другим подразделением нашей компании.

Это решение позволило существенно повысить реализм симуляции, а также повысить ценность тренажеров для обучения персонала, непосредственно сталкивающегося с подобной техникой. Придать уверенности в работе с настоящим оборудованием.

Разработка сценариев с использованием транспорта потребовала кардинально нового подхода. В частности, в проекте было реализовано нестандартное решение, которое отсутствует в базовом наборе Unreal Engine, для имитации работы техники с разделением на переднюю и заднюю полураму.

Данный комплекс тренажеров включает уникальный набор интерактивных упражнений, максимально приближенных к реальным условиям и оборудованию: обучение отгрузке горной массы, отработка технологии бурения шпуров, моделирование закладки взрывных смесей

Для реализации задуманного была создана система модульного навесного оборудования с независимым управлением как виртуальным, так и физическим устройством.

Проект стал важным шагом в развитии нашей VR-платформы и открыл новое направление в создании тренажеров, позволяющих эффективно готовить персонал к работе в сложных и потенциально опасных условиях подземного производства.
