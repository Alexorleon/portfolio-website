ProExpVR360

В промежутке между основными проектами поступила задача оперативно разработать VR-приложение для воспроизведения видео 360°. Главной особенностью стало полное исключение взаимодействия пользователя с интерфейсом: запуск ролика, авторизация и назначение контента происходили автоматически, без дополнительных действий со стороны зрителя.

  • Released: 2021
  • Platforms: Android (Oculus Quest)
  • Team Size: on my own
  • Duration: <1 month
  • Responsibilities: геймдизайн, разработка, дизайн уровня
  • Tools Used: Unreal Engine 4, C++, Blueprints, Git, Postman

Description

Detailed Info

Система работает следующим образом: все настройки и выбор видео происходят на стороне браузера, после чего пользователю остается только надеть очки — видео начинается автоматически. По завершению просмотра программа переходит в режим ожидания для подготовки к новому сеансу.

Проект был рассчитан на короткий срок выполнения, и одной из ключевых задач стало взаимодействие с уже существующим веб-сервером, разработанным другой командой. Мы провели детальное обсуждение форматов обмена данными и интеграции с сервером, чтобы обеспечить стабильную и быструю работу приложения.

Благодаря продуктивному сотрудничеству и четко выстроенному процессу разработки приложение было успешно реализовано.
