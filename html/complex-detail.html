Предсменное экспресс-обучение

Предсменное экспресс-обучение - это процесс обучения и тестирования работников перед началом смены по профильным вопросам с демонстрацией последствий от неправильных действий для повышения компетентности работника. Оценка компетенции и профессиональных рисков работников с формированием отчетности и статистики для руководителей.

  • Released: 2017
  • Platforms: Windows | Linux
  • Team Size: four core members
  • Duration: 3 years
  • Responsibilities: руководство командой разработчиков, создание серверной и клиентской части кода, проектирование базы данных, разработка клиентского кроссплатформенного приложения
  • Tools Used: Qt, C++, QML, PHP, HTML/CSS/JS, Oracle, Git, OpenCV

Description

Detailed Info

Работник решает задачу, которая состоит из иллюстрации, вопроса к задаче и трех вариантов ее решения. При правильном решении демонстрируются опасности, которые он избежал. При неправильном решении запускается процедура управления риском - корректировка компетентности.

Формируется аналитика и оценка профессиональных рисков по результатам экзаменатора. Ежемесячно, система направляет данные, характеризующие состояние компетентности и прогноз количества травмированных.

Приложение разрабатывалось с учетом модульности и возможности формирования только необходимой группы устройств.

В августе 2017 г. специалисты Горной Секции Международной ассоциации социального обеспечения пришли к заключению, что технология «Кузбасс-ЦОТ» отвечает критериям программ «Профилактика несчастных случаев на рабочих местах. Нулевое видение в области травматизма» и присудили методике «Кузбасс-ЦОТ» Знак Качества «Превосходство Горной Секции МАСО».
